# 🎮 Reaction Rush

**Reaction Rush** — мобильная игра на скорость реакции, разработанная на React Native.  
Игроку необходимо успевать нажимать на появляющиеся точки до их исчезновения.  
Сложность игры можно регулировать в настройках.

---

## 🚀 Возможности

- 🎯 Динамическая игровая механика (появляющиеся точки)
- 🔥 Комбо-система с множителем очков
- ⚙️ Настройки сложности (Easy / Medium / Hard)
- 🌍 Поддержка двух языков (RU / EN)
- 💾 Сохранение настроек через Zustand + AsyncStorage
- 🎨 Градиентный игровой фон
- ✨ Анимация появления точек
- 🎵 Фоновая музыка в главном меню
- 🏗 Архитектура FSD (Feature-Sliced Design)

---

## 📱 Экраны приложения

### 🟣 Welcome Screen
Стартовый экран с анимацией логотипа и автоматическим переходом в главное меню.

### 🟢 Menu Screen
Главное меню приложения:
- ▶ Play — начать игру  
- ⚙ Settings — открыть настройки  
- 📖 Rules — посмотреть правила игры  

### 🎮 Game Screen
Игровой экран:
- Появляющиеся точки
- Таймер раунда (30 секунд)
- Подсчёт очков
- Подсчёт промахов
- Комбо-система (множитель очков)
- Экран результатов после окончания раунда

### ⚙ Settings Screen
Экран настроек:
- 🌍 Выбор языка (Русский / English)
- 🎯 Выбор сложности (Easy / Medium / Hard)

Все настройки сохраняются при перезапуске приложения.

### 📖 Rules Screen
Экран с описанием правил игры:
- Дождаться появления точки
- Успеть нажать до её исчезновения
- Не допускать промахов

---

## 🧠 Игровая механика

- Раунд длится 30 секунд
- Максимально допустимо 3 промаха
- Если точка исчезает без нажатия — засчитывается промах
- Комбо увеличивает множитель очков:
  - 5 попаданий подряд → x2
  - 10 → x3
  - 15+ → x4

---

## 🏗 Архитектура проекта

Проект построен по методологии **Feature-Sliced Design (FSD)**:


- **app** — навигация и инициализация  
- **pages** — экраны приложения  
- **features** — пользовательские действия (смена языка, сложности)  
- **entities** — бизнес-сущности (settings)  
- **shared** — переиспользуемые компоненты, тема, утилиты

# 🖥 Установка и запуск

## 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/reactionRush.git
cd reactionRush
```

## 2. Клонировать репозиторий

```bash
npm install
```

## 3. Установить iOS зависимости

```bash
cd ios
pod install
cd ..
```

## 4. Запустить приложение (iOS)

```bash
npx react-native run-ios
```

# ⚠️ Возможные проблемы

## Native module not found

```bash
cd ios
rm -rf Pods Podfile.lock
pod install
cd ..
```

## Проблемы со сборкой iOS

```bash
cd ios
xcodebuild clean
cd ..
```
